# richard-yelpcamp
 The website built on MEN stack. The deployed URL is <https://richard-yelpcamp.herokuapp.com/>

The whole project is guided by Udemy course (https://www.udemy.com/course-dashboard-redirect/?course_id=625204)

## Built With

- MongoDB - The cloud database is provided by Mongo Atlas
- Express - The backend is written by express
- NodeJS - The language used.
- Passport - Used to authenticate and authorize users
